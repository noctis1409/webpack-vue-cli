# webpack-vue-cli

使用 webpack4 搭建的 vue 脚手架。

## 目前使用的 loader

1. style-loader
2. css-loader
3. postcss-loader
4. file-loader
5. babel-loader
6. vue-loader

## 目前使用的 plugin

1. html-webpack-plugin
2. clean-webpack-plugin

## 其他技术

1. 使用 webpack-dev-server 搭建开发者服务器。
2. 使用 webpack-merge 分离 webpack 配置文件。
